# rust-sqlx-research

This was research done by [GeekMasher](https://github.com/GeekMasher) to see if SQLx has the same issues as other frameworks in other languages.

## Setup

1. Creating a `.env` file with
  - `DATABASE_URL`
  - `POSTGRES_USER`
  - `POSTGRES_PASSWORD`
2. Export or set the `DATABASE_URL`
